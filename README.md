# 📂 Projeto Gestão Documental - Análise de Dados

## Sobre o projeto

Este projeto aplica análise de dados na área de Gestão Documental, utilizando Python para gerar, analisar e visualizar informações sobre documentos.

O objetivo é criar indicadores que auxiliem no acompanhamento documental, permitindo visualizar status, distribuição por setores e geração de relatórios.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Streamlit
- Excel

## Funcionalidades

✅ Geração de base de dados documental  
✅ Análise exploratória dos dados  
✅ Criação de gráficos  
✅ Cálculo de indicadores  
✅ Geração de relatório Excel  
✅ Dashboard interativo

## Indicadores analisados

- Quantidade total de documentos
- Documentos ativos
- Documentos arquivados
- Documentos pendentes
- Distribuição por setor
- Tipos de documentos
